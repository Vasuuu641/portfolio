# Portfolio Website

A personal portfolio website built with **React**, **TypeScript**, and **Tailwind CSS**.  
This project is a work in progress and showcases my skills as a **full-stack developer**.

---

## Features

- Hero section  
- About section  
- Projects section  
- Responsive layout  
- Smooth scrolling navigation  

---

## Tech Stack

- React  
- TypeScript  
- Tailwind CSS  
- Vite (recommended)

---
